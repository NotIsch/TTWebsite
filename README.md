# King's School Tech Team Portal

Welcome to the official repository for the **King's School Tech Team Portal**.

The portal is designed to be the central hub for the school's Tech Team, providing communication, event management, technical documentation, and resources for both teachers and Tech Team members.

---

## Features

### 🔐 Role-Based Login

Users sign in using their school account and are assigned one of two roles:

* 👨‍🏫 **Teacher**
* 🎛️ **Tech Team Member**

Each role has access to a customised dashboard and features relevant to their needs.

---

## 👨‍🏫 Teacher Dashboard

Teachers can quickly request technical support for school events.

### Booking System

Teachers can submit requests for:

* School productions
* Assemblies
* Concerts
* Open evenings
* Drama rehearsals
* Presentations
* Guest speakers
* Award ceremonies

Each booking includes:

* Event name
* Date
* Start and finish time
* Location
* Event type
* Lighting requirements
* Sound requirements
* Projection requirements
* Additional notes

The system displays Tech Team availability and prevents conflicting bookings.

## Project Goal

The King's School Tech Team Portal aims to bring together all of the tools and information needed by the school's technical theatre team into one easy-to-use platform. It streamlines communication between teachers and technicians, simplifies event bookings, and provides comprehensive documentation for lighting, sound, and visualisation systems used throughout the school.
